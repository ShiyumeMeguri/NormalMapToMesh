# Normal Map To Mesh (法线 → 多级精度高模)

把切线空间法线贴图重建为 Multires(多级精度)修改器上的真实高模几何细节,
观感等价于**渲染时法线贴图的逐像素光滑效果**——不是模糊出来的, 是解析算出来的。

## 用法

1. 选中目标网格(需有 UV)。`3D视图 > 侧栏(N) > Ruri > 法线 → 多级精度高模`。
2. **游戏资产**(材质里已有法线贴图节点链): 来源保持"自动", 直接点 **应用 / 更新**。
3. **裸网格 + 贴图文件**: 点 **加载法线并一键构建** 选文件(自动切到贴图模式)。
4. 调 **高度倍数**(默认 1.0 = 物理高度)后再点应用——从基面幂等重建, 不叠加。
5. **移除细节** 一键恢复低模。

工作分辨率**不需要设置**: 自动等于实际接入的法线贴图原生分辨率(沿材质 Normal Map
节点的 Color 输入网络回溯找到的 Image Texture 尺寸)。设更高只是把已有像素插值
放大、不产生新细节还多耗算力; 设更低则白白丢弃贴图里的信息——两者都没有意义,
所以没有暴露成选项。

## 原理

**烘焙从未被需要**: 材质里的法线贴图信息是逐像素/逐材质节点求值出来的, 不需要
Cycles 参与。数学核心——把切线空间法线贴图转成高度场:

```
坡度(切平面向量, |g|=tanθ):  g = N − T'/(T'·N),  T' = 材质法线链算出的扰动法线
高度梯度:                    dh/du = g·∂P/∂u,  dh/dv = g·∂P/∂v
物理高度场:                  Frankot-Chellappa 频域泊松积分(O(N logN))
```

`T`(切线)、`B=sign·N×T`(副切线)、`N`(法线)、`∂P/∂u,∂P/∂v`(表面对 UV 的解析偏导)
全部来自 **mikktspace 切线帧**(`calc_tangents`, 与渲染器同源)+ 逐三角形解析几何,
在 UV 网格上向量化光栅化, 天然携带镜像 UV 岛的手性(负绕向翻转)、逐岛任意旋转的
切线朝向、逐岛不同的 texel 世界密度——这些是 v1(直接拿贴图梯度当高度场全局积分,
假设"UV 轴=切线轴")在真实资产(实测发型: 406 UV 岛 / 144 镜像岛)上失败的地方。

**shader 等价求值纪律(锯齿的真正根源)**: 这个发型的正/背面卡片、镜像复用 UV
共用同一批 texel。把切线帧光栅化到"一张共享网格"上, 每个 texel 只能存一个
"赢家"面的属性——正/背面高度函数互为相反数, 赢家按三角形光栅化顺序拼成马赛克,
积分后即是**逐三角形补丁的 ±h 锯齿**, 任何模糊都救不了(模糊 ±h 拼图只会拼出更难看
的抵消纹)。渲染器从不遇到这个问题, 因为每个面永远用**自己的**插值属性求值,
贴图才是唯一共享物。因此:
- 切线帧标量只由**正 UV 绕向**面贡献且逐 texel **平均**(孤儿镜像岛用负绕向补洞),
  不做覆盖式光栅化;
- 位移方向、基面曲率(见下)全部改为在**低模自身**上做解析插值, 不进任何共享网格。

**渲染级光滑的完整几何化**(缺一不可, 每一层都对应渲染 shader 里真实存在的一步):

| 渲染里做什么 | 这里怎么几何化 |
| --- | --- |
| 顶点法线逐像素 Phong 插值 | 位移方向 = 基面 loop 法线在表面参数点的重心插值(面内 C∞) |
| 平滑法线暗示的弯曲表面 | **真正的 Catmull-Clark 细分**(全局迭代平均, 收敛到 C2 光滑极限曲面) |
| 纹理逐像素连续采样 | 高度场用三次 B 样条(C2)采样——双线性(C0)在 texel 边界导数跳变, 正是"素模视图颗粒感"的来源 |
| — (采样定理硬约束) | 顶点密度稀于 texel 时的高斯预滤**下限**(纯抗混叠, 非风格化模糊; 自动级别下=0) |

**曾走过的弯路(Phong Tessellation, 已废弃)**: 一度尝试用线性基面 + 逐三角形解析
鼓形(`α·(p*−p)`, `p*=Σwᵢ·proj_{Nᵢ}(p)`, PN 三角/Boubekeur–Alexa 方案)替代真正的
细分曲面, 理由是"CC 的极限形状不是着色暗示的那个曲面"。实测在真实资产上翻车:
Phong 鼓起是逐三角形**独立**的局部方案, 只保证共享边位置连续(C0), 不保证曲率
连续; 该发型的平滑法线场(非自定义, 就是 Blender auto-smooth 算出来的)相邻三角形
间常有数十度量级的独立偏差, 每个三角形各自把这份偏差鼓成几何, 大量三角形密集
重复即刻出明显的表面皱纹。CC 细分是全局迭代平均, 没有这个"逐面独立决策"的病理,
即使输入法线场本身有较大局部变化也不会有.现已完全移除 Phong Tessellation 相关代码,
基面曲率统一交给 Subsurf 的真实 Catmull-Clark/Simple/Linear 细分(`细分方式`选项)。

**边界处理**: Catmull-Clark 会把开放边界向内收缩(细分自身的漂移, 与位移无关)——
用同拓扑 SIMPLE 对照细分把边界顶点拉回基面线性边缘原位, 校正量经 smoothstep +
图扩散做成连续 C1 混合场(边界=0/内部=1), 边界钉死不产生卡片缝隙, 也不会因为
整数环线性渐变在环边界留下导数跳变(那会沿所有卡片边缘拉出脊线)。位移向量场
同样在边界严格归零, 内部图拉普拉斯平滑。

**逐岛去趋势(平面拟合) + 岛间最小二乘缝合**: 全局积分给每个 UV 岛留下的任意
常量偏移/线性斜坡, 用基面拓扑映射(细分面按基面连续分块, 不怕 UV 重叠)精确归岛
后消除, 长发/部件分段处不再有高度台阶。

**材质法线链免烘焙求值**: 游戏资产常见的通道重建网络(TexImage/Separate/
Combine/Math/VectorMath/NormalMap 等)有一个小型 numpy 节点求值器直接执行;
遇到不支持的节点/网格(非三角四边面等)自动整体回退 Cycles EMIT 烘焙(见下)。

### 保留的回退路径: Cycles EMIT 烘焙

直算前端要求材质节点在求值器支持范围内。遇到不支持的节点时, 整个前端回退到
物体空间 EMIT 烘焙(注意 Blender 5.x 的 NORMAL 烘焙通道不含材质法线扰动, 实测
逐位相同, 只能用 EMIT 发射色编码), 同一套梯度/积分/岛处理代数继续走下去——
只是数据来源换成三张烘焙图(扰动法线/基准法线/表面位置)。烘焙期间临时切
Cycles(CPU)、隐藏其它物体渲染、插入烘焙目标节点, 结束后全部还原; 链接库材质
无法插烘焙节点, 请先 Make Local。

### 性能纪律

- **Multires 建层只为数据结构**(reshape 会完整覆写位移层), 隐藏态跑
  `multires_subdivide`; 细分基面统一走 **Subsurf 求值副本**(拓扑/顶点序与
  multires 求值逐位一致, 实测), 不受 Mesh 里已有位移数据影响。CC 模式额外
  按指纹缓存一份 SIMPLE 对照细分坐标(边界校正用)。
- **重建快路径**: 层数已匹配时跳过删层重细分, 前端/对照细分/岛处理全缓存命中
  ——调倍数重建实测与首建逐位相同(max diff = 0)。
- 每次 `bpy.ops` 调用都触发整场景深度图刷新, 重型角色场景每算子白算十余秒——
  构建期间把无关物体临时摘出视图层, 结束后还原。
- 全程 numpy 向量化(光栅化按三角形 bbox 尺寸分桶批处理), 零逐顶点 Python 循环。

## 选项

| 选项 | 说明 |
| --- | --- |
| 来源 | 自动(材质优先)/ 材质(求值现有材质法线输出)/ 贴图(临时节点链) |
| 高度倍数 | 1.0 = 积分出的物理高度(物体空间单位); 平贴=0; 负值反向 |
| 细分方式 | 平滑(Catmull-Clark, 默认, 圆润有机形体)/ Simple(保形, 完全不平滑)/ Linear |
| 边缘/位移平滑 | 位移场图拉普拉斯迭代次数; 边界顶点始终锁死零位移, 0=只锁边 |
| 平整死区 (LSB) | 法线 XY 低于该 8bit 台阶数视为纯平, 防量化噪声积分放大成斑点 |
| 坡度上限 | 坡度(tanθ)限幅, 压制噪声/压缩伪影导致的孤立尖刺 |
| 自动级别 / 四边形预算 | 级别自动匹配工作分辨率(源贴图原生) × UV 占用率, 预算兜底内存 |
| 强制烘焙路径 | 跳过直算前端, 强制走 Cycles EMIT 烘焙(兼容任意节点网络) |

## 注意

- 构建期间会**临时**把无关物体摘出视图层求值——结束后全部还原, 不留痕迹。
- 直算前端要求网格为三角/四边形(`calc_tangents` 限制), 含五边以上面自动回退烘焙。
- 贴图模式会把文件图色彩空间自动置为 Non-Color(生成型图像跳过, 防清像素)。
- 物体上已有**非本工具创建**且带层级的 Multires 时会拒绝执行(防细节丢失)。

## 开发

`core.py` 为纯 numpy 数学内核, 不依赖 bpy:

```bash
python core.py   # 自测: 双线性/B样条采样 / 端到端高度还原(旋转·异尺度·镜像岛) /
                  # 直算光栅化(覆盖+累加平均) / FC锚定 / UV岛并查集 / 去趋势 / 缝合
```
