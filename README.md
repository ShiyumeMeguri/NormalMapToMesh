# Normal Map To Mesh (法线 → 多级精度高模)

把一张与当前选中网格 UV 完全对应的**切线空间法线贴图**,重建为 Multires(多级精度)修改器上的真实高模几何细节。一键全自动:加载贴图 → 自动细分 → 位移 → 写回,可随时调强度重建。

## 用法

1. 选中目标网格(需有 UV)。
2. `3D视图 > 侧栏(N) > Ruri > 法线 → 多级精度高模`。
3. 点 **加载法线并一键构建**,选文件,完事。
4. 调 **强度** 后点 **应用 / 更新** —— 从基面幂等重建,不会叠加。
5. **移除细节** 一键恢复低模。

## 原理

- 法线解码为梯度场 `(-nx/nz, -ny/nz)`,经 **Frankot-Chellappa 频域泊松积分**(FFT,O(N logN))重建高度场;按指纹缓存,调强度重跑近乎零成本。
- 强度 `1.0` = 按低模 **texel 世界密度**(√(表面积/UV面积))推算的物理高度;标准闭环验证(Blender 烘焙圆顶→重建)RMS 0.12mm、峰值误差 0.03%。
- **基准面锚定**:零梯度平坦区锚定为 0 高度(全图零均值会被凸起把平地压负,即"整体下沉"问题的根因)。
- **量化死区**:±1 LSB 内的法线视为纯平 —— 8bit 量化噪声经积分会放大成低频起伏。
- **三项全自动约定检测**:V 方向(有效像素命中投票)、绿通道 OpenGL/DirectX(可积性旋度判据)、凹凸方向(高度偏度先验),均可手动覆盖。
- 自动级别:`四边形数 ≈ 贴图texel数 × UV占用率`,受四边形预算(默认 1600 万)保护。
- 细分默认 **Simple**(保形),位移沿细分面顶点法线。
- **UV 岛处理**(图集贴图核心):面级并查集找岛 → 细分面按基面连续分块的**拓扑精确映射**(不怕 UV 重叠)→ 逐岛去趋势(平面拟合/中位数锚定)→ 岛间最小二乘**缝合**(共享顶点约束,消分段台阶)。
- 写回用 `multires_reshape`,细节存进 Multires 位移层,可继续雕刻/烘焙。
- 全程 numpy 向量化 + `foreach_get/set`,无逐顶点 Python 循环。

## 选项

| 选项 | 说明 |
| --- | --- |
| 细分方式 | Simple(保形,推荐)/ Catmull-Clark / Linear |
| 自动级别 / 四边形预算 | 级别自动匹配 texel 密度,预算兜底内存 |
| 去趋势 | 平面(图集必开)/ 均值 / 关 |
| 凹凸方向 | 自动(凸起优先)/ 原样 / 反转 |
| V 方向 | 自动 / 正常 / 翻转(D3D 约定资产) |
| 绿通道 | 自动(旋度判据)/ OpenGL / DirectX |
| 高通截止 (px) | 抑制跨岛低频鼓包(图集残留时设 128~512) |
| 平滑 (px) | 高斯低通,压噪点/锯齿 |
| 平整死区 (LSB) | 量化噪声死区,默认 1.0 |
| 坡度上限 | 梯度限幅,压烘焙噪声/压缩伪影尖刺 |
| 双面卡片背面反向 | 自动识别共面反法线的背面副本(UV 负绕向 + BVH 共面验证),位移取反;镜像复用 UV 与筒状前后两层不受影响 |
| 重建 Z 通道 | BC5/AG 双通道贴图从 XY 重建 Z |

## 烘焙须知(法线贴图从哪来)

要让重建"无限逼近"原始高模,烘焙本身必须正确:

1. **源物体与目标低模必须空间重叠**(selected-to-active 靠射线求交,分开摆放=垃圾数据/黑斑)。
2. **cage_extrusion 必须大于源物体最大凸起高度**,否则凸起顶部射线打空、被填平(实测:0.05 外扩烘 0.214 高圆顶 → 只重建出 37% 高度;0.30 外扩 → 99.97%)。
3. 优先浮点/16bit 贴图;8bit 也可,量化噪声由死区兜底。

## 注意

- 文件贴图的色彩空间会自动置为 Non-Color(生成型图像跳过——改色彩空间会清掉烘焙像素)。
- 物体上已有**非本工具创建**且带层级的 Multires 时会拒绝执行(防细节丢失),请先应用或移除。
- alpha=0(通道非全空时)/ 长度异常 / 纯黑的像素视为无效,按平地处理。
