# Normal Map To Mesh (法线 → 多级精度高模)

把切线空间法线贴图重建为 Multires(多级精度)修改器上的真实高模几何细节。
v4 = **免烘焙直算**: mikktspace 切线帧光栅化 + numpy 材质法线链求值装配高度梯度,
频域泊松积分出物理高度, 沿基准法线位移。平贴严格零位移、不膨胀、不破面,
一键全自动, 倍数可调; 不支持的材质节点自动回退 Cycles EMIT 烘焙。

## 用法

1. 选中目标网格(需有 UV)。`3D视图 > 侧栏(N) > Ruri > 法线 → 多级精度高模`。
2. **游戏资产**(材质里已有法线贴图节点链): 来源保持"自动", 直接点 **应用 / 更新**。
3. **裸网格 + 贴图文件**: 点 **加载法线并一键构建** 选文件(自动切到贴图模式)。
4. 调 **高度倍数**(默认 1.0 = 物理高度)后再点应用——从基面幂等重建, 不叠加。
5. **移除细节** 一键恢复低模。

## 原理(三代方案的教训与最终解)

- **v1(已废弃)** 直接按"UV 轴 = 切线轴"解读贴图梯度做全局积分。真实游戏资产
  (实测发型: **406 个 UV 岛 / 144 个镜像岛 / 逐岛任意朝向的切线帧 / 混合手性**)
  逐岛梯度方向错乱——前提性失败。
- **v2(已废弃)** 差分位移 (n1−n0)×scale: 平贴零位移达成, 但幅值只随倾斜角、
  与细节**波长无关**——高频发丝沟壑的位移超过顶点间距, 表面撕碎。
  坡度必须先积分成高度(高频自动得到小高度)才物理正确。
- **v3** 用 Cycles **EMIT 烘焙三张物体空间图**(n1/n0/P)零猜测装配梯度——正确但慢,
  且 Blender 5.x 的 NORMAL 烘焙通道不含材质法线扰动(实测逐位相同), 只能走 EMIT
  发射色编码。现保留为**回退路径**(材质网络含直算不支持的节点时自动启用)。
- **v4 直算前端(默认)**: 同一代数免烘焙——mikktspace 切线帧(calc_tangents,
  与渲染器同源)+ 角法线 + 逐三角形解析 ∂P/∂u,∂P/∂v 光栅化到 UV 网格(numpy
  分桶向量化); 材质法线链(游戏资产常见的通道重建网络)用 numpy 节点求值器直接算
  (TexImage/Separate/Combine/Math/VectorMath/NormalMap 等); 梯度
  dh/du = −[(T·∂P/∂u)·tx + (B·∂P/∂u)·ty]/max(tz, 0.2|t|)。
  镜像/混合手性由逐 loop bitangent_sign 精确携带; 解析 ∂P 消灭了位置图有限差分
  与岛间沟槽假差分; 梯度只取真实 UV 覆盖区(外扩 margin 的复制内容不入积分),
  法线通道单独外扩保证岛边界采样连续。与烘焙路径在同一资产上幅值统计吻合(±2%)。
- **Frankot-Chellappa 频域泊松积分**(O(N logN)) → 物理高度场: 平贴处梯度严格 0
  → 高度 0(平地锚定, 无整体膨胀); 高频细节获得与波长匹配的小高度, 不破面。
- **逐岛去趋势(平面拟合) + 岛间最小二乘缝合**(共享顶点约束): 消除全局积分留给
  每岛的任意偏移/斜坡/台阶。岛归属用基面拓扑映射(细分面按基面连续分块), 不怕
  UV 重叠。双面卡片正背面共享 texel → 同一高度、沿各自基准法线同向跟随。
- 细分默认 **Simple**(保形); 位移经 `multires_reshape` 写入 Multires 位移层,
  可继续雕刻/再烘焙。
- **边缘锁定 + 位移场平滑**: 开放边界(卡片边缘)顶点位移严格归零——边缘偏移会把
  原本贴合的卡片边撕出缝隙, 基面边缘本来就正确(等价沿边缘刷"擦除多级精度置换");
  位移向量场再做图拉普拉斯平滑(边界 Dirichlet 0), 位移向边缘平滑衰减并去高频斑点。
- **细分统一走 Subsurf 求值副本**: 拓扑/顶点序与 multires 求值逐位一致(实测),
  SIMPLE+uv_smooth=NONE 即精确线性细分——细分基面与 CC 对照都不再依赖 multires
  求值/算子, 也天然不受 Mesh 里已有 MDISPS 影响(v3.2 对照继承旧层空转 bug 的
  结构性根除)。multires 建层只为数据结构(隐藏态跑 subdivide), reshape 完整覆写。
- **平滑细分的边界校正**: Catmull-Clark 会把开放边界向内收缩(锁位移锁不住细分
  自身的漂移)——把边界顶点拉回线性细分位置(基面边缘原位), 校正沿图距离在
  2^(L-1) 环内线性衰减融入平滑内部; 对照坐标按指纹缓存。
- **重建快路径**: 层数匹配时跳过删层重细分, 前端/对照/岛全缓存命中——调倍数
  重建实测与首建逐位相同(max diff = 0), CC 24.8s / SIMPLE 16.6s。
- **算子阶段性能纪律**: 把无关物体临时摘出视图层求值(每次 bpy.ops 都触发整场景
  深度图刷新, 重型角色场景每算子白算十余秒), 结束后还原。
  历代耗时(同资产 CC 全新构建): v3.2 190s → v3.3 94s → v4 81s(重建 25s)。
- 逐 loop 双线性采样 + 逐顶点平均(接缝顶点取两侧均值), 全 numpy 向量化。
- 三张烘焙图按(网格指纹 × 材质 × 来源 × 分辨率)会话内缓存, 调倍数重跑近乎零成本;
  构建完成会报告岛数与位移幅值 p50/p95/max。

## 选项

| 选项 | 说明 |
| --- | --- |
| 来源 | 自动(材质优先)/ 材质(烘焙现有材质法线输出)/ 贴图(临时节点链) |
| 高度倍数 | 1.0 = 积分出的物理高度(物体空间单位); 平贴=0; 负值反向 |
| 烘焙分辨率 | 512~4096, 通常取源法线贴图同档(默认 2048) |
| 边缘/位移平滑 | 拉普拉斯迭代次数; 边界顶点始终锁死零位移, 0=只锁边 |
| 细分方式 | Simple(保形, 推荐)/ 平滑(Catmull-Clark, 带边界校正)/ Linear |
| 自动级别 / 四边形预算 | 级别自动匹配 texel 密度, 预算兜底内存 |
| 强制烘焙路径 | 跳过直算前端, 强制 Cycles EMIT 烘焙(兼容任意节点网络) |

## 注意

- 构建期间会**临时**把无关物体摘出视图层求值——结束后全部还原, 不留痕迹。
- 回退烘焙路径运行时会临时: 切 Cycles(CPU)、隐藏其它物体渲染、往材质插烘焙节点,
  同样全部还原; 链接库材质无法插烘焙节点, 请先 Make Local。
- 直算前端要求网格为三角/四边形(calc_tangents 限制), 含五边以上面自动回退烘焙。
- 贴图模式会把文件图色彩空间自动置为 Non-Color(生成型图像跳过, 防清像素)。
- 物体上已有**非本工具创建**且带层级的 Multires 时会拒绝执行(防细节丢失)。

`core.py` 为纯 numpy 数学内核, 不依赖 bpy:

```bash
python core.py   # 自测: 双线性/端到端还原(旋转·异尺度·镜像岛)/直算光栅化/FC锚定/UV岛/缝合
```
