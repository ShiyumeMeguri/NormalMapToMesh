# Normal Map To Mesh (法线 → 多级精度高模)

把切线空间法线贴图重建为 Multires(多级精度)修改器上的真实高模几何细节,
观感等价于**渲染时法线贴图的逐像素光滑效果**——不是模糊出来的, 是解析算出来的。

## 用法

1. 选中目标网格(需有 UV)。`3D视图 > 侧栏(N) > Ruri > 法线 → 多级精度高模`。
2. **游戏资产**(材质里已有法线贴图节点链): 来源保持"自动", 直接点 **应用 / 更新**。
3. **裸网格 + 贴图文件**: 点 **加载法线并一键构建** 选文件(自动切到贴图模式)。
4. 调 **高度倍数**(默认 1.0 = 物理高度)后再点应用——从基面幂等重建, 不叠加。
5. **移除细节** 一键恢复低模。

工作分辨率**不需要设置**: 自动等于实际接入的法线贴图原生分辨率(沿材质 Normal Map
节点的 Color 输入网络回溯找到的 Image Texture 尺寸)。设更高只是把已有像素插值
放大、不产生新细节还多耗算力; 设更低则白白丢弃贴图里的信息——两者都没有意义,
所以没有暴露成选项。

## 原理

**v6 不变式(架构基石)**: **一切场量只由低模 + 贴图决定, 细分只是对同一连续
曲面的采样密度**。任何细分级别都输出同一张曲面(`use_limit_surface`: 各级顶点
都落在同一张极限曲面上, 嵌套采样); 若输出随细分漂移, 说明管线在某处读了细分
网格自身的数据(离散角法线/逐级坐标), 即为架构错误。

**基面 = 岛界折痕锁定的 Catmull-Clark 极限曲面(v6.1)**: 法线贴图只携带高频
细节, 低模的粗曲率必须由光滑细分补上(线性细分会把低模折面原样留在高模上)。
但 CC 默认把开放边界/折痕链平滑成 B 样条曲线——卡片边缘被软化、彼此贴合的边
撕出缝隙。因此对**所有 UV 岛边界**(开放边、跨岛边、UV 接缝边)施加 crease=1 +
岛界顶点 vertex crease=1: 折痕链逐级只插线性中点、原顶点钉死, 边界折线被
**精确锁在低模原位**(等价 EdgeCrease 的严格版), 内部收敛到 C2 极限曲面。
折痕写在求值副本上并与用户已有折痕取 max, 原网格不动。

**烘焙从未被需要**: 数学核心——把切线空间法线贴图转成高度场:

```
坡度(切平面向量, |g|=tanθ):  g = N − T'/(T'·N),  T' = 材质法线链算出的扰动法线
高度梯度:                    dh/du = g·∂P/∂u,  dh/dv = g·∂P/∂v
物理高度场:                  镜像 Neumann 最小二乘泊松积分(O(N logN))
```

`T`(切线)、`B=sign·N×T`(副切线)、`N`(法线)、`∂P/∂u,∂P/∂v`(表面对 UV 的解析偏导)
全部来自 **mikktspace 切线帧**(`calc_tangents`, 与渲染器同源)+ 逐三角形解析几何,
在 UV 网格上向量化光栅化, 天然携带镜像 UV 岛的手性(负绕向翻转)、逐岛任意旋转的
切线朝向、逐岛不同的 texel 世界密度——这些是 v1(直接拿贴图梯度当高度场全局积分,
假设"UV 轴=切线轴")在真实资产(实测发型: 406 UV 岛 / 144 镜像岛)上失败的地方。

**镜像 Neumann 积分(v6)**: 旧周期 FFT 假设高度场在图集边界环绕连续——实际图集
非周期, 岛间隔着边界互相泄漏低频(此前靠高通滤波压制, 是拐杖且顺带削真实起伏)。
v6 把梯度按对称性半样本镜像到 2H×2W(gx 沿 x 反对称/沿 y 对称, gy 相反)再做
周期 Frankot-Chellappa, 对称数据的最小二乘解自动继承偶对称, 取原象限即 Neumann
边界的精确解: 非周期斜坡可精确还原, 零 wrap-around, 高通拐杖废除。

**shader 等价求值纪律(锯齿的真正根源)**: 发型类资产的正/背面卡片、镜像复用 UV
共用同一批 texel。把切线帧光栅化到"一张共享网格"上, 每个 texel 只能存一个
"赢家"面的属性——正/背面高度函数互为相反数, 赢家按三角形光栅化顺序拼成马赛克,
积分后即是**逐三角形补丁的 ±h 锯齿**, 任何模糊都救不了。渲染器从不遇到这个问题,
因为每个面永远用**自己的**插值属性求值, 贴图才是唯一共享物。因此:
- 切线帧标量只由**正 UV 绕向**面贡献且逐 texel **平均**(孤儿镜像岛用负绕向补洞),
  不做覆盖式光栅化;
- 位移方向不进任何共享网格, 由低模自身的插值场求值(见下)。

**渲染级光滑的完整几何化**(每一层都对应渲染 shader 里真实存在的一步):

| 渲染里做什么 | 这里怎么几何化 |
| --- | --- |
| 平滑法线暗示的粗曲率(光照里的"圆") | 岛界折痕锁定的 **Catmull-Clark 极限曲面**(全局迭代平均, C2, 无逐面独立病理), 边界折线钉死原位 |
| 顶点法线逐像素重心插值 + normalize | 低模平滑角法线写成 corner 属性随细分**线性插值**(uv_smooth=NONE), 插值后归一化——位移方向场面内 C∞, 逐位 shader 同构 |
| 纹理逐像素连续采样 | 高度场用三次 B 样条(C2)采样——双线性(C0)在 texel 边界导数跳变, 正是"素模视图颗粒感"的来源 |
| — (采样定理硬约束) | 顶点密度稀于 texel 时的高斯预滤**下限**(纯抗混叠, 非风格化模糊; 自动级别下=0) |

**曾走过的弯路(基面方案史)**: v5 的 Phong Tessellation(逐三角形 PN 鼓起)——
逐面独立方案只保证 C0, 真实资产平滑法线场的逐面偏差被各自鼓成密集皱纹, 废弃;
v5.3 的裸 CC——边界被平滑软化, 靠"SIMPLE 对照细分 + 环形混合"事后拉回, 是
band-aid 且输出随细分方式漂移; v6.0 的纯 SIMPLE——边界精确但低模折面原样留在
高模上(法线图只有高频, 补不了粗曲率)。v6.1 定稿: **CC 极限曲面提供曲率 +
岛界折痕在细分方案内部锁死边界**, 两个目标在同一个细分算子里同时成立, 无
事后校正。

**边界处理(细分不变)**: 几何层——岛界折痕把基面边界折线精确锁在低模原位
(见上, 细分方案内部成立, 无事后校正); 位移层——高度场在 UV 域沿开放边的距离场
做 smoothstep 衰减归零(半径按贴图像素), 边界顶点另有硬锁零兜底。两层合起来:
彼此贴合的卡片边在高模上仍严格贴合, 零缝隙。衰减场属于低模 UV 域, 与细分级别
无关(旧方案在细分网格拓扑上做图拉普拉斯平滑, 半径随级别漂移, 违反不变式, 已废除)。

**逐岛去趋势(平面拟合) + 岛间最小二乘缝合**: 全局积分给每个 UV 岛留下的任意
常量偏移/线性斜坡, 用基面拓扑映射(细分面按基面连续分块, 不怕 UV 重叠)精确归岛
后消除, 长发/部件分段处不再有高度台阶。

**材质法线链免烘焙求值**: 游戏资产常见的通道重建网络(TexImage/Separate/
Combine/Math/VectorMath/NormalMap 等)有一个小型 numpy 节点求值器直接执行;
遇到不支持的节点/网格(非三角四边面等)自动整体回退 Cycles EMIT 烘焙(见下)。

### 保留的回退路径: Cycles EMIT 烘焙

直算前端要求材质节点在求值器支持范围内。遇到不支持的节点时, 整个前端回退到
物体空间 EMIT 烘焙(注意 Blender 5.x 的 NORMAL 烘焙通道不含材质法线扰动, 实测
逐位相同, 只能用 EMIT 发射色编码), 同一套梯度/积分/岛处理代数继续走下去——
只是数据来源换成三张烘焙图(扰动法线/基准法线/表面位置), 三张图全部烘自低模,
不破坏 v6 不变式。烘焙期间临时切 Cycles(CPU)、隐藏其它物体渲染、插入烘焙目标
节点, 结束后全部还原; 链接库材质无法插烘焙节点, 请先 Make Local。

### 性能纪律

- **Multires 建层只为数据结构**(reshape 会完整覆写位移层), 隐藏态跑
  `multires_subdivide`; 细分基面统一走 **Subsurf 折痕 CC 极限求值副本**(拓扑/
  顶点序与 multires 求值逐位一致, 实测), 不受 Mesh 里已有位移数据影响。
- **重建快路径**: 层数已匹配时跳过删层重细分, 前端/岛处理全缓存命中——
  调倍数重建实测与首建逐位相同(max diff = 0)。
- 每次 `bpy.ops` 调用都触发整场景深度图刷新, 重型角色场景每算子白算十余秒——
  构建期间把无关物体临时摘出视图层, 结束后还原。
- 全程 numpy 向量化(光栅化按三角形 bbox 尺寸分桶批处理), 零逐顶点 Python 循环。
- 镜像积分在 4K 工作分辨率下瞬时内存 ~1.5GB(numpy FFT 固定 complex128)。

## 选项

| 选项 | 说明 |
| --- | --- |
| 来源 | 自动(材质优先)/ 材质(求值现有材质法线输出)/ 贴图(临时节点链) |
| 高度倍数 | 1.0 = 积分出的物理高度(物体空间单位); 平贴=0; 负值反向 |
| 边缘衰减 (px) | 高度场向开放边界的 smoothstep 距离衰减半径(贴图像素); 边界顶点始终硬锁零位移; 0=只硬锁边 |
| 平整死区 (LSB) | 法线 XY 低于该 8bit 台阶数视为纯平, 防量化噪声积分放大成斑点 |
| 坡度上限 | 坡度(tanθ)限幅, 压制噪声/压缩伪影导致的孤立尖刺 |
| 自动级别 / 四边形预算 | 级别自动匹配工作分辨率(源贴图原生) × UV 占用率, 预算兜底内存 |
| 强制烘焙路径 | 跳过直算前端, 强制走 Cycles EMIT 烘焙(兼容任意节点网络) |

## 注意

- 构建期间会**临时**把无关物体摘出视图层求值——结束后全部还原, 不留痕迹。
- 直算前端要求网格为三角/四边形(`calc_tangents` 限制), 含五边以上面自动回退烘焙。
- 贴图模式会把文件图色彩空间自动置为 Non-Color(生成型图像跳过, 防清像素)。
- 物体上已有**非本工具创建**且带层级的 Multires 时会拒绝执行(防细节丢失)。

## 开发

`core.py` 为纯 numpy 数学内核, 不依赖 bpy:

```bash
python core.py   # 自测: 双线性/B样条采样 / 端到端高度还原(旋转·异尺度·镜像岛) /
                  # 直算光栅化(覆盖+累加平均) / Neumann斜坡还原 / 锚定 / 边缘衰减 /
                  # UV岛并查集 / 去趋势 / 缝合
```
